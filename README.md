## Sobre o projeto

Este portfólio foi construído do zero com HTML, CSS e JavaScript puros — sem frameworks —
como exercício consciente de dominar os fundamentos antes de abstrações.

Está em refatoração ativa: o objetivo da próxima versão é componentização com maior
separação de responsabilidades e melhoria de UI/UX.
